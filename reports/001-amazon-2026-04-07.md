**2. Invent & Simplify: Tell me about a creative technical solution**

- **T:** Brute force was infeasible; needed a fast, correct solver

**3. Deep Dive: Tell me about working with messy/unreliable data (Deep Dive LP)**

- **T:** Extract structured data reliably enough for client-facing quality control reports
- **A:** Wrote Python parsers using PDFMiner/pdfplumber with regex + heuristic fallbacks; validated against known ground truth for each PDF variant
- **R:** Fully automated extraction pipeline — engineering teams got reports in seconds vs. hours of manual work
- **R:** Would build an ML-based extractor for truly novel layouts to reduce heuristic maintenance

**4. Learn and Be Curious: Tell me about learning a new technology quickly**

- **R:** Live pipeline running with real-time threat scoring and Streamlit dashboard
