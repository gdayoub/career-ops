**2. Tell me about a data pipeline you built (Aligns with Affirm intern project examples)**

- **R:** Replaced hours of manual work per cycle; used directly in client-facing QC deliverables
- **R:** Would add a data quality dashboard for field-by-field extraction confidence scores

**3. Tell me about working independently with minimal guidance**

- **T:** Ramp up and deliver results in 10 weeks
- **A:** Reverse-engineered the codebase, set up my own dev environment, wrote documentation as I learned, prioritized the highest-value pipeline first, and shipped weekly
- **R:** Delivered full pipeline in 4 weeks; weeks 5–10 added SQL reporting and visualizations
- **R:** Learned that documentation-while-doing is faster than documentation-after

**4. Tell me about a time you handled a technical failure (Resilience)**

- **A:** Diagnosed the session expiry issue using network devtools; refactored the Selenium auth flow to handle session refresh; added retry logic with exponential backoff
- **R:** Scraper ran stably for all subsequent development and deployment
- **R:** Added session health checks to the startup routine to catch similar failures early

**5. Why Affirm? Why fintech? (Motivation)**
